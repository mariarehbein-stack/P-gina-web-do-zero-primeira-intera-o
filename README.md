Evolução do Cultivo do Café no Paraná

Projeto desenvolvido utilizando HTML, CSS e JavaScript.

☕ Tema do Projeto
📌 Sobre o Projeto

O projeto apresenta, de forma visual, interativa e educativa, a história, a evolução e o desenvolvimento do cultivo do café no estado do Paraná.

A proposta mostra como a cafeicultura foi uma das principais responsáveis pelo crescimento econômico do estado, impulsionando a ocupação do Norte do Paraná, o surgimento de cidades, o fortalecimento da agricultura e a modernização das técnicas de produção ao longo das décadas.

Por meio de textos, imagens, linha do tempo, acordeão interativo, quiz e um jogo, o site busca aproximar o visitante da história da cafeicultura paranaense, destacando sua importância econômica, social, cultural e tecnológica para o Paraná e para o Brasil.

🎯 Objetivo

Conscientizar os visitantes sobre a importância histórica e econômica da cafeicultura no Paraná, mostrando como a evolução das técnicas agrícolas permitiu aumentar a produtividade, preservar o meio ambiente e fortalecer o desenvolvimento rural.

O site busca também valorizar a memória da cafeicultura paranaense e facilitar o acesso ao conhecimento por meio de uma plataforma digital moderna e interativa.

🌱 Protótipo do jogo do trator

Crie o jogo do zero seguindo as seguintes características: O personagem principal deve ser um emoji de trator 🚜. Os obstáculos devem ser elementos relacionados à agricultura posicionados aleatoriamente no caminho, porém com um intervalo para que apareçam de forma fluida, com no máximo dois obstáculos por vez. O fundo do jogo deve ser inspirado em uma grande plantação de café vista de cima, com tons naturais, áreas cultivadas, terra e vegetação. O cenário deve se mover continuamente para criar sensação de velocidade e progresso. O jogador deve controlar o trator desviando dos obstáculos utilizando as teclas S/W ou setas esquerda/direita. O jogo deve aumentar gradualmente a velocidade conforme o tempo passa. Adicione sistema de pontuação baseado no tempo sobrevivido. Inclua tela de "Game Over" com botão de reiniciar. O jogo deve iniciar com um botão de início. Utilize canvas ou elementos HTML animados para criar a movimentação do jogo. O estilo visual deverá seguir a temática da cafeicultura. O jogo deverá ser estilo horizontal, incluindo três faixas por onde o trator poderá se movimentar.
